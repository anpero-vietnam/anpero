Aceren Windows Mixed Virtual Reality is one of the first products in Acer’s Windows Mixed Reality ecosystem.

This Aceren Windows Mixed virtual reality mirror does not use removable sensors to record motion. Therefore, it will bring the feeling of neatness, easy installation and moving. Instead of using removable sensors, the Aceren Windows Mixed virtual glasses utilize two built-in cameras on the headset.
To use the software, Windows Mixed uses the Xbox controller (Xbox Controller) to control.

Detailed configuration of the glass:

Resolution: 1440 x 1440
Lens size: 2.89 inches x 2 eyes
FOV view: 95 degrees
Scanning frequency: 90Hz
Built-in audio and microphone: Output via 3.5 mm headphone jack
Connect to PC: HDMI 2.0 and USB 3.0
Cable length: 4 meters
